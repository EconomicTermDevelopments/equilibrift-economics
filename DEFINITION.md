# Equilibrift: Technical Definition

    ## Formal Definition
    Market state where a theoretical clearing equilibrium exists but becomes dynamically unreachable due to adaptation lags or capacity constraints that create insurmountable gaps between current system state and equilibrium position.

    ## Conceptual Background
    Equilibrift is used here as an operational concept rather than a rhetorical label. It addresses market conditions where equilibrium exists in theory but remains dynamically unreachable. The mechanism focus is adjustment delays and capacity constraints prevent state variables from converging to clearing conditions, which can be represented with explicit channels: equilibrium_gap, adjustment_lag, capacity_constraint, coordination_failure, expectation_dispersion, policy_response_delay, adaptive_capacity. This repository couples concept definition with executable implementation so assumptions remain inspectable. In practice, the framework supports comparative statics, scenario simulation, and policy diagnostics. Typical applications include agent-based model diagnostics, market-stability monitoring, and policy timing evaluation. Because the index logic is transparent, researchers can vary weights and test robustness across model specifications.
